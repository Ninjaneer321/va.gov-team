## 2 - Appointment Reminder Notifications

- [Research folder](https://github.com/department-of-veterans-affairs/va.gov-team/tree/master/products/va-notify/research/appointment-reminder)

### Methods
- Content review and synthesis
- Notification analysis across VA systems

### Summary
Review, consolidate, and write appointment notifications in plain language. Veterans receive a lot of notifications from many different sources. The goal is to review all notifications that Veterans receive about appointments, conduct user testing to see which are most effective, define the events and triggers for notifications, agree on plain language content, and design how notifications will be surfaced to Veterans.
