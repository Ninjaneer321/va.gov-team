## Summary

The VA Notify API experienced an outage of approximately 4.5 hours due to a database issue. Following API restoration, SMS delivery remained degraded for an additional ~7.3 hours. Full resolution occurred within the same calendar month.

## Root Cause

Database issue causing API unavailability. Full root-cause analysis available in [vanotify-team/Incidents/README.md](https://github.com/department-of-veterans-affairs/vanotify-team/blob/main/Incidents/README.md).

## Resolution

API service was restored after approximately 4.5 hours. SMS delivery was separately restored approximately 7.3 hours after the API came back online. Full resolution details available in [vanotify-team/Incidents/README.md](https://github.com/department-of-veterans-affairs/vanotify-team/blob/main/Incidents/README.md).
