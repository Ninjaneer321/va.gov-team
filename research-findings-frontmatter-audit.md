## Methodology

### File Detection
Files were identified as research findings reports if their filename (case-insensitive) contained any of the following patterns:

- `research-findings`
- `research-report`
- `findings-report`
- `research-results`
- `topline-findings`
- `topline-report`
- `topline`
- `usability-findings`
- `usability-report`
- `discovery-findings`

Variations with spaces or underscores instead of hyphens were also matched (e.g., `ResearchFindings.md`, `research_findings.md`).

### Frontmatter Analysis
A file was flagged as having **no frontmatter** if it did not begin with a `---` delimited YAML block.

A file was flagged as having **placeholder frontmatter** if the YAML block contained any of:
- Generic template values (e.g., `Product Name`, `Team Name`, `YYYY-MM-DD`)
- Placeholder researcher names (e.g., `Researcher/Designer 1`)
- Template goals/methods (e.g., `Goal 1`, `Method 1`)
- Multiple `X` placeholder values for numeric fields
